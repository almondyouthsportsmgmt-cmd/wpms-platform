# WPMS Release 0.9 — Employees & Permissions

## Added
- Employee directory
- Add and edit employee
- Owner, Manager, Groomer, Bather, Boarding Attendant, and Receptionist roles
- Role-based default permissions
- Per-employee permission overrides
- Active/inactive status
- Hire date and hourly rate tracking
- Calendar color assignment
- Employee search and filters
- Demo-mode persistence
- Supabase employee table and RLS

## Test checklist
1. Add an employee.
2. Change the role and verify permission defaults.
3. Override individual module permissions.
4. Edit hourly rate and status.
5. Search by name, email, phone, and role.
6. Filter by role and status.
7. Refresh and confirm persistence.
