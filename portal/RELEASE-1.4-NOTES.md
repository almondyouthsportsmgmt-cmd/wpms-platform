# WPMS Release 1.4 — Staff Mobile Workspace

## Added
- Mobile-first staff task board
- Employee selector and per-employee task queue
- Clock-in and clock-out tracking
- Grooming, boarding, pickup, medication, and cleaning tasks
- Pending, In Progress, and Completed workflow
- Pet, customer, time, location, and task notes
- Demo-mode persistence
- Supabase mobile task and shift-session tables

## Test checklist
1. Open Staff Mobile from the sidebar.
2. Select an employee.
3. Clock in and clock out.
4. Start and complete tasks.
5. Refresh and confirm demo persistence.
6. Verify the page at phone width.
