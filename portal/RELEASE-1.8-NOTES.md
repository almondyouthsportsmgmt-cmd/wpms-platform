# WPMS Release 1.8 — Automated Reminders & Follow-Ups

## Added
- Appointment confirmation and reminder automation
- Boarding check-in and check-out reminders
- Vaccination-expiration follow-ups
- Payment follow-up reminders
- Membership renewal reminders
- We-miss-you customer re-engagement
- SMS, email, and combined delivery channels
- Scheduled, sent, failed, skipped, and cancelled statuses
- Send-now and cancel actions
- Customer and pet targeting
- Search and filtering
- Demo-mode persistence
- Supabase-ready automated-reminder table

## Migration hardening
- Corrected Release 1.7 membership policies to be idempotent.
- All Release 1.8 policies and triggers can be safely rerun.
