# WPMS Release 2.0 — Kennel Map & Capacity Planner

## Added
- Interactive kennel and suite directory
- Zone-based organization
- Available, reserved, occupied, cleaning, and maintenance statuses
- Customer and pet assignment
- Check-in and check-out visibility
- Capacity tracking
- Search and status filters
- Demo-mode persistence
- Idempotent Supabase migration

## Test checklist
1. Open Kennel Map.
2. Add a kennel or suite.
3. Assign a customer and pet.
4. Change status between available, reserved, occupied, cleaning, and maintenance.
5. Search by kennel, zone, pet, or owner.
6. Refresh and confirm persistence.
