# WPMS Release 1.3 — Two-Way SMS Integration Foundation

## Added
- SMS provider configuration
- Android phone bridge configuration
- Existing business-number support
- Two-way message transport foundation
- Appointment reminders
- Boarding updates
- Ready-for-pickup notifications
- Quiet hours
- Connection test workflow
- SMS delivery event table
- Demo-mode persistence
- Provider adapter support for Twilio and Telnyx

## Test checklist
1. Open Two-Way SMS.
2. Select Android Bridge.
3. Enter the shop number, device name, and bridge endpoint.
4. Test the connection.
5. Save and refresh.
6. Toggle each automation.
7. Confirm settings persist.
