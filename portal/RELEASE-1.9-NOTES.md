# WPMS Release 1.9 — Client Forms & Digital Agreements

## Added
- Grooming consent forms
- Boarding agreements
- Medication authorizations
- New client intake forms
- Vaccination waivers
- Customer and pet assignment
- Draft, sent, signed, declined, and expired statuses
- Signature tracking
- Expiration tracking
- Search and filters
- Demo-mode persistence
- Idempotent Supabase migration

## Test checklist
1. Create a digital form.
2. Send the form.
3. Mark it signed.
4. Expire an unsigned form.
5. Filter by type and status.
6. Refresh and confirm persistence.
